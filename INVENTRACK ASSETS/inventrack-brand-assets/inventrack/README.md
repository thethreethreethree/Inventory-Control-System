# InvenTrack Solutions — Brand Asset Package

Vector-first identity system rebuilt from your source logo. Every mark is a
clean SVG (infinitely scalable, editable) with PNG exports at common sizes.

## Brand colors
| Token        | Hex      | RGB           | Use |
|--------------|----------|---------------|-----|
| Deep Navy    | #0B2D50  | 11, 45, 80    | Wordmark, primary surfaces |
| Steel Blue   | #1E4D7B  | 30, 77, 123   | Gear accent, secondary |
| Track Blue   | #2E6DB4  | 46, 109, 180  | Highlights, links, tagline periods |
| Ink Black    | #0A0A0A  | 10, 10, 10    | Gear body, body text |

## Typography
- **Wordmark / headings:** Arial Black (or Montserrat ExtraBold) — all caps, +1 tracking
- **Body / UI:** Arial (or Inter / Open Sans) — sentence case
- **Tagline:** caps, +3 tracking, periods in Track Blue

## Quick usage notes
- Web favicon: use favicon.svg (modern browsers) with favicon.ico fallback;
  apple-touch-icon = favicon-180.png.
- Social profiles: social-avatar-512.png.
- Print/large format: always use the SVG or the 2400px PNG.
- On photos or dark UI: use logo-reversed; on single-color constraints: mono.
- Minimum size: horizontal lockup 140px wide; icon 24px; favicon down to 16px.

All assets derive from the same gear geometry, so they stay perfectly consistent.
